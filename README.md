## Description

The Voting System is a Python-based application that allows voter registration, secure vote casting, and election result calculation. The system prevents duplicate voting and ensures that only registered voters can participate.

This project demonstrates user authentication, data validation, vote collection, and result tabulation.



## Features

* Voter registration
* User authentication
* Vote casting
* Duplicate vote prevention
* Candidate selection
* Real-time result display
* Menu-driven interface

## How to Run

1. Install Python 3.
2. Download or clone the repository.
3. Open Terminal or Command Prompt.
4. Run:

python voting_system.py

5. Register voters, cast votes, and view results.
